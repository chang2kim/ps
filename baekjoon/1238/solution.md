# 백준 골드 3 - 파티

https://www.acmicpc.net/problem/1238

## 단서

- 가장 거리가 먼 학생 -> 각 학생별 최단거리가 필요
- 마을 -> X -> 마을 : 정방향, 역방향의 다익스트라
- 정방향 다익스트라는 X -> 마을, 일반적으로 u -> v, weight 정보를 사용
- 역방향 다익스트라는 마을 -> X
- u -> ? 의 정보를 키로 하여 graph.get(u) 인접 리스트를 그린다면,
  ? -> v의 정보를 키로 하여 graph.get(v) 인접 리스트로 역방향 처리가능
- 학생마다 distFromX + distToX를 처리 후 가장 큰 녀석이 정답
